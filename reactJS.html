ReactJS

ReactJS è una libreria JavaScript open-source utilizzata per la creazione di interfacce utente (UI) dinamiche e interattive. È stata sviluppata da Facebook e ha guadagnato molta popolarità nella comunità degli sviluppatori web.

Introduzione

React è una libreria per la creazione di siti web su singola pagina. I siti React si aggiornano in background senza dover ricaricare la pagina. È adatto per gestire registri, magazzini, ordinazioni e social network. Offre modularità e la possibilità di importare moduli per funzionalità aggiuntive come la gestione di PDF o la creazione di grafici.

Installazione

  • Controllo della versione dell'npm 

          
            npm -v

Potremmo riscontrare un problema con la versione dell'npm che ci impedisce di creare un nuovo progetto.
Soluzione: 

          
            sudo npm install -g n
          
          
            sudo n stable



Creazione primo progetto

  • Installazione del progetto base 

          
            npm create-react-app nomeProgetto

  • Avvio del progetto base scaricato 

          
            npm start

  • Cancellare il contenuto del file "App.js".

  • Cancellare il contenuto del file "App.css"

  • Rimuovere i tag lasciando solo per evitare problemi con i controlli

  • Eliminare tutti i file di test

Target



Attraverso i target ci è possibile restituire il valore di un elemento selezionato, tramite una funziona

          
            import React, {useRef} from 'react'
                      
            // creazione della nostra applicazione
            const App = () => {

              // funzione cliccami
              const cliccami =(event) => {
                // alert collegato al div
                alert(event.target.textContent)
              }

              // funzione cliccami 2
              const cliccami2 =(event) => {
                // alert nome
                alert(event.target.dataset.nome)
              }

              // stampa a video
              return (
                <div>
                  {/* Event target */}
                  <div onClick={cliccami}>
                    <p>Hola</p>
                    <div>Ciao</div>
                    <p>Bonjour</p>
                  </div>

                  <div class="p" onClick={cliccami2} className="card-img">
                    {/* alle immagini aggiungo un attributo personalizzato "data-nome" */}
                    <img data-nome = "Ducati Panigale V4 S" src="https://hd2.tudocdn.net/1064315?w=1920" width="400vh" alt=""/>
                    <img data-nome = "Ducati Multistrada" src="https://smanettoni.net/wp-content/uploads/2021/10/Ducati-Multistrada
                    -V4-Pikes-Peak-2022-01.jpg" width="350vh" alt=""/>
                    <img data-nome = "Ducati Hypermotard" src="https://www.ducatimilano.it/wp-content/uploads/2018/11/Hypermotard-SP
                    -MY2021-Dinamiche-14-gallery-1920x1080.jpg" width="300vh" alt=""/>
                  </div>
                </div>
              )
            }

            export default App
